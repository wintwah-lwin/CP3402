## Content Insights
* Set up a local environment creates a virtual sandbox without incurring cloud costs like a live server host.
* Run PHP via an Apache local web server enables server-side execution such as database requests, manages user session states (`session_start()`)
* Configure relational database tables (`questions`, `usertb`, `attempts`) in phpMyAdmin and connecting them through `dbconnect.php` are necessary for the dynamic site database connection without sql injection attack
