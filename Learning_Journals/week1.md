## Content Insights
*   Using Bootstrap’s grid system is faster and more consistent than writing custom code for the site.
*   I struggled with the navigation bar failing to collapse. I learned this was a simple syntax error—my `id` and `data-bs-target` values did not match.
*   Separating custom CSS from Bootstrap defaults keeps the codebase clean.

## Career/Employability/Learning Insights
*   Mastering frameworks like Bootstrap is more employable than building every component manually since it is faster and nmore consistent.
*   I spent an hour "guessing" solutions to my navbar issue. If I am blocked for more than 15 minutes, I must immediately pivot to official documentation, and aid from community like Reddit, or asking AI to troubleshoot.
*   Moving code from `localhost` to a live server is a critical skill. I need to get comfortable with more live hosting environments outside of Github pages as well.
