## Content Insights
*   Moving from a static HTML site to a CMS like Joomla was a significant shift in logic. In Prac 1, I had full control over every file structure. In Joomla, the flow is completely different. The content is stored in articles and the template structures are in modules.
*   The biggest challenge was template customization. I initially tried to edit core files directly, the changes weren't reflected as I hoped for when I run it live. So, for some design features I had to rely on custom css codes to get the exact layout that I wanted.

## Career/Employability/Learning Insights
*   Working with Joomla means less coding and is faster to deploy than custom codes.
*   I noticed that the learning curve for Joomla is steeper than I expected compared to when I was experimenting with Wordpress. This exercise has made me realize that I need to be versatile with different platforms because I cannot guarantee a future employer will stick to one specific technology.
*   I feel more confident in my ability to navigate a new software interface, but I am also realizing that "simple" CMS sites can become complex very quickly if you do not fully understand how the basics and layout works.
